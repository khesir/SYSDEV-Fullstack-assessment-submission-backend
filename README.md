### Run this project

Here's the environment variables, create .env and follow this format
```
PORT = 3000

MYSQL_HOST = '<host>'
MYSQL_USER = '<root>'
MYSQL_PASSWORD = '<password>'
MYSQL_DATABASE = '<database>'
```

Setup the project
```
npm install
```

Run this project
```
npm start
```
